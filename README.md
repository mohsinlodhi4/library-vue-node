# library-vue-node

<h3>How To Run Library Management system </h3>

Step 1: run server app <br/>
run the following commands: <br/>

cd library-server-node && npm i <br/>
npm run dev <br/>

Step 2: run client app <br/>
run the following commands: <br/>

cd .. <br/>
cd library-client-vue && npm i <br/>
npm run dev <br/>
